üìñ Property Definitions

  • Reflexive: (a,a) ‚àà R for every a ‚àà A
  • Symmetric: If (a,b) ‚àà R, then (b,a) ‚àà R
  • Transitive: If (a,b) ‚àà R and (b,c) ‚àà R, then (a,c) ‚àà R
  • Equivalence: Reflexive + Symmetric + Transitive

üü∞ Equivalence Relations

Equivalence Relations

A relation is an equivalence relation if it is reflexive, symmetric, and transitive. Equivalence relations partition sets into equivalence classes.
