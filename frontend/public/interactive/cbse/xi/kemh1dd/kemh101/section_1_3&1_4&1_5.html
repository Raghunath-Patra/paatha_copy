The Empty Set (φ or { })

A set which does not contain any element is called the empty set, null set, or void set.

Symbol: φ or { }

Example 1: No Natural Numbers

φ

Set: {x : 1 < x < 2, x ∈ ℕ}

Why Empty: No natural number exists between 1 and 2

φ = { }

Example 2: Irrational Solution

φ

Set: {x : x² - 2 = 0, x ∈ ℚ}

Why Empty: √2 is irrational, not rational

φ = { }

Example 3: Even Prime > 2

φ

Set: {x : x is even prime > 2}

Why Empty: 2 is the only even prime number

φ = { }

Example 4: Impossible Condition

φ

Set: {x : x² = 4, x is odd}

Why Empty: Solutions are ±2 (both even)

φ = { }

Finite and Infinite Sets

Finite Set: A set with a definite number of elements (or empty)

Infinite Set: A set with unlimited number of elements

Finite Set Examples

5

Days of week: {Mon, Tue, Wed, Thu, Fri, Sat, Sun}

n(Set) = 7 (finite)


Solutions of x² = 16: {-4, 4}

n(Set) = 2 (finite)

Infinite Set Examples

Equal Sets

Definition: Two sets A and B are equal if they have exactly the same elements

Notation: A = B (equal) or A ≠ B (not equal)

Key Point: Order and repetition don't matter!

Equal Sets Example 1

Set A: {1, 2, 3, 4}

Set B: {3, 1, 4, 2}

A = B ✓

Same elements, different order

Equal Sets Example 2

Set A: {1, 2, 3}

Set B: {2, 2, 1, 3, 3}

A = B ✓

Repetition doesn't change a set

Not Equal Sets

Set A: {1, 2, 3}

Set B: {1, 2, 4}

A ≠ B ✗

Different elements: 3 vs 4
