What is a Set?

A set is a well-defined collection of objects. The objects in a set are called elements or members of the set.

Standard Number Sets

ℕ Natural Numbers: {1, 2, 3, ...}

ℤ Integers: {..., -2, -1, 0, 1, 2, ...}

ℚ Rational Numbers

ℝ Real Numbers

ℤ⁺ Positive Integers

ℚ⁺ Positive Rationals

ℝ⁺ Positive Reals

Set Membership

∈ means "belongs to" or "is an element of"

∉ means "does not belong to"


Example: If V = {a, e, i, o, u}

Then: a ∈ V (a belongs to V)

But: b ∉ V (b does not belong to V)

Set Relationships

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ


ℕ (Natural Numbers): {1, 2, 3, 4, ...}

ℤ (Integers): {..., -2, -1, 0, 1, 2, ...}

ℚ (Rational Numbers): Numbers that can be expressed as p/q where p, q ∈ ℤ and q ≠ 0

ℝ (Real Numbers): All rational and irrational numbers


⊂ means "is a subset of"

Every natural number is also an integer, every integer is also rational, and every rational number is also real.

Two Ways to Represent Sets

1. Roster Form: List all elements separated by commas within braces { }

2. Set-Builder Form: Describe the common property of elements
