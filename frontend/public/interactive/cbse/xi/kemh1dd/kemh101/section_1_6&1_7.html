Subsets (⊂)

Definition: A set A is a subset of set B if every element of A is also an element of B

Notation: A ⊂ B (read as "A is a subset of B")

Key Facts: φ ⊂ every set, Every set ⊂ itself

Counting numbers starting from 1

ℤ (Integers)

ℤ = {..., -2, -1, 0, 1, 2, ...}

Natural numbers + 0 + negative numbers

Examples: -5, 0, 8

ℚ (Rational Numbers)

ℚ = {p/q : p,q ∈ ℤ, q ≠ 0}

Numbers expressible as fractions

Examples: 1/2, -3/4, 0.75

ℝ (Real Numbers)

All rational + irrational numbers

All numbers on the number line

Examples: π, √2, e, -√5

Universal Sets

Definition: A universal set U is the set that contains all elements under consideration in a particular context

Notation: Usually denoted by U, and all other sets are subsets of U

Example 1: Number Theory

Universal Set: ℝ (Real Numbers)

Subsets: ℕ, ℤ, ℚ, etc.

When studying number properties

Example 2: School Context

Universal Set: All students in school

Subsets: Students in each class

When analyzing student data

Example 3: Playing Cards

Universal Set: Standard deck (52 cards)

Subsets: Hearts, spades, face cards

When studying card games

Example 4: Geometry

Universal Set: All points in a plane

Subsets: Triangles, circles, polygons

When studying geometric shapes
